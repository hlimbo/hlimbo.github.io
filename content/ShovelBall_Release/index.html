Controls:

Player 1 (Blue Player):

  • Jump - W key
  • Move Left - A key
  • Move Right - D Key
  • Swing Shovel - Spacebar

Player 2 (Red Player):

  • Jump - up arrow key
  • Move Left - left arrow key
  • Move Right - right arrow Key
  • Swing Shovel - Period (.)/Greater than Symbol (>) Key

About Game:

ShovelBall is a 2 player local game where you compete against another to score the most points in the opposing player's goal before time runs out. The game's physics was inspired by Super Smash Bros. and its gameplay may remind you of Rocket League.

This was made during the Winter Quarter 2015 Game Jam at UCI for the Video Game Development Club (VGDC). This was my first game jame I participated in and was very excited yet unsure of how I would help develop the game alongside with my randomly selected group. Thankfully, I was grouped with an amazing programmer who helped show me the ropes on how to collaborate as a team. He introduced me to git version control, taught common coding practices in C# during game jams, and encouraged me to write scripts to respawn balls when they were hit towards the goal as well as the physics code for the ball movement. Overall, the game jam was a fun learning experience and I hope everyone who plays this game enjoys it as much as I did when I helped develop it.

FAQ:

1. My game won't load anymore. How do I fix this?

Make sure you have no other tabs open in your browser besides this game to ensure game runs without errors. If that doesn't work, close the web browser and reopen it.

2. My game window stopped responding or I can't see what's happening in the game anymore.

The game was designed to be played when your web browser is fully maximized. If the browser was resized, you may need to refresh your web browser to properly play the game.

3. Screen sizes?

The game should be able to support all computer resolutions. When viewing this page on mobile or tablet, the page layout may become all jumbled up.
